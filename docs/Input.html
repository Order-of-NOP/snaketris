Class: Input

Input()

Input manager that doesn't mind to be used with pads. Translates controller actions (e.g. KeyCode.W) into terms of gameplay actions (e.g. player1.jump).

Constructor

new Input()

To be called on game create. Uses Phaser's game.input objects.
